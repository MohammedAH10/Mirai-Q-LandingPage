# Mirai-Q

## AI-Powered Learning & Collaboration Ecosystem for African Education

Mirai-Q is a nonprofit social enterprise building the future of learning for Africa through AI-powered education, collaboration, and digital access.

### Getting Started

```bash
# Install dependencies
npm install

# Start the frontend dev server
npm run dev

# Start the backend waitlist server (in a separate terminal)
npm run server

# Or run both together
npm run dev:all
```

### Building for Production

```bash
npm run build
```

The built files will be in the `dist/` directory.
